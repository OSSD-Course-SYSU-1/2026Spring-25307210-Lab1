# 任务3：新增功能说明文档

## 1. 新增功能
账单删除功能

## 2. 功能描述
在账单列表的每条记录右侧，添加红色删除按钮。点击按钮后，该条账单会立即从列表中移除，同时顶部的总收入和总支出金额会自动重新计算并更新。

## 3. 修改文件
- 文件路径：entry/src/main/ets/components/shturl.cc/Uhu8q
- 新增 `deleteBill` 方法，实现账单删除和金额重新计算逻辑
- 在账单列表项中添加可点击的删除按钮组件

## 4. 运行效果
1.  打开APP首页，账单列表按日期正常分组显示
2.  每条账单右侧显示红色【删除】按钮
3.  点击删除按钮，对应账单立即从列表中消失
4.  顶部的总收入和总支出金额自动更新
5.  所有原有记账、统计功能不受影响
